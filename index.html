Static positions are per html flow.These elements are considered to be not positioned.
Relative position is the same as static unless you give it extra values.
Giving values to top, right bottom and left will move the element. These values are relative to its default position.
Behaves like relative except that it is relative to its parent element. Top, right, bottom, and left give it a newposition.
Fixed position will move to a new position and stay there even if the content around it is being scrolled through.
